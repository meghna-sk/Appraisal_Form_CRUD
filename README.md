# Appraisal-Form
  A single-page application for filling Employee Appraisal Forms. Can perform all CRUD operations
## Tools used
  1. MongoDB
  2. NodeJS
  3. ExpressJS
  4. AngularJS
## How to Run
  In the directory run the following commands:
  1. `npm install`
  2. `npm start`
  3. Website runs at localhost:3000
